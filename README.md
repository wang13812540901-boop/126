# AI视频生成项目

这个项目包含了AI视频生成工具和相关的入门指南。

## 文件说明

- `video_generator.py` - 使用OpenCV生成基础视频的Python脚本
- `index.html` - 前端开发入门指南
- `jimeng_video_generator.html` - 即梦AI视频生成界面
- `sessionid_guide.html` - Chrome浏览器获取SessionID指南

## 如何使用

### 运行Python视频生成器

```bash
python3 video_generator.py
```

### 查看HTML文件

直接在浏览器中打开对应的HTML文件即可。

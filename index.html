什么是前端开发？

前端开发是创建网站或应用程序用户界面的过程，主要使用以下技术：

  • HTML - 网页的结构
  • CSS - 网页的样式
  • JavaScript - 网页的交互功能

第一个示例：Hello World

<!DOCTYPE html> <html> <head> <title>Hello World</title> </head> <body> <h1>Hello World!</h1> </body> </html>

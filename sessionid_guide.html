Chrome浏览器获取SessionID详细指南

步骤1：登录即梦官网

打开Chrome浏览器，访问 https://jimeng.jianying.com 并登录你的账号。

步骤2：打开开发者工具

在Chrome浏览器中，有三种方法打开开发者工具：

  • 方法1：按键盘快捷键 Option + Command + I (Mac) 或 Ctrl + Shift + I (Windows)
  • 方法2：点击浏览器右上角的三个点 → 更多工具 → 开发者工具
  • 方法3：在页面上右键点击，选择「检查」

步骤3：找到Application标签

在开发者工具的顶部标签栏中，找到并点击 Application 标签。如果没有看到这个标签，点击标签栏右侧的 » 按钮，然后选择 Application。

注意：在Chrome的新版本中，这个标签可能叫做 Storage。

步骤4：找到Cookies

在左侧导航栏中，展开 Cookies 选项，然后点击显示的网站地址（应该是 jimeng.jianying.com）。

步骤5：复制sessionid

在右侧的表格中，找到 sessionid 这一行，然后复制 Value 列中的值。

提示：你可以双击Value列中的值，然后按Command+C (Mac) 或 Ctrl+C (Windows) 来复制。

步骤6：粘贴到视频生成界面

回到 jimeng_video_generator.html 页面，将复制的sessionid粘贴到「即梦SessionID」输入框中。

如果仍然找不到

如果以上步骤仍然无法找到sessionid，你可以尝试以下方法：

  • 在开发者工具中切换到 Network 标签
  • 刷新页面
  • 在请求列表中找到一个请求，点击它
  • 在右侧的 Headers 标签中，找到 Request Headers 部分
  • 查找 Cookie 字段，其中包含sessionid
